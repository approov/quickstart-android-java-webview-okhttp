Android WebView + Approov

Remote calls stay inside the native Approov path.

This quickstart page uses normal JavaScript fetch(), but the transport is replaced by the reusable Approov WebView helper package. The page never sends the Shapes api-key directly and never needs access to the Approov SDK.
